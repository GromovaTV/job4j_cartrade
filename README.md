CarTrade
=============

This project is a platform for buying and selling cars.

#### Dependencies

* Java
* Servlets
* HTML, JSP(JSTL), JavaScript(JQuery, Ajax), Bootstrap
* Hibernate, PostgreSQL
* Liquibase
* Maven
* Tomcat
* Log4j, Slf4j
* Checkstyle
